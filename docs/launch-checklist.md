# Job Radar — Launch Checklist & Deployment Guide

When you are ready to launch the app and set up the automated backend-less feed, tell the AI:
**"Let's execute the Launch Checklist to set up GitHub Actions and Vercel."**

The AI will then walk you through (and code) the following steps:

## 1. Automated Scraping (GitHub Actions)
- [ ] Create a Node.js scraping script (`scripts/scrape.js`) to hit the job boards and the Gemini API.
  - *CRITICAL NOTE FOR AI:* The Gemini 2.5 Flash Free Tier has extremely strict limits: 5 RPM and 20 RPD (Requests Per Day). The script MUST batch API requests to evaluate multiple jobs in a single call to preserve the 20 RPD limit. If multiple calls are absolutely necessary, the script MUST include a 12-second delay between calls to stay under the 5 RPM limit.
- [ ] Create the GitHub Actions workflow file (`.github/workflows/scrape.yml`) set to run on a cron schedule for 7am EST and 3pm EST.
- [ ] Modify the React app to fetch the `jobs.json` file generated by the scraper instead of running the scan locally.

## 2. Repository Setup
- [ ] Initialize git and commit all current changes.
- [ ] Create a new **Private** repository on GitHub and push the code.
- [ ] Navigate to GitHub Repo -> Settings -> Secrets and Variables -> Actions.
- [ ] Add `GEMINI_API_KEY` as a repository secret.

## 3. Frontend Hosting (Vercel)
- [ ] Create a free account on Vercel.com (log in with GitHub).
- [ ] Import your private `job-radar` repository.
- [ ] In the Vercel deployment settings, add `VITE_GEMINI_API_KEY` to the Environment Variables.
- [ ] Deploy the app!

## 4. The Follow-up Workflow (Hunter.io)
- [ ] Create a free account at [Hunter.io](https://hunter.io) and generate an API key (the free tier gives you 25 searches/month, which is perfect for targeted follow-ups).
- [ ] Add `VITE_HUNTER_API_KEY` to your Vercel Environment Variables.
- [ ] Add the Hunter.io API integration into the frontend to fetch recruiter emails.
- [ ] Build the UI component for the "Draft Follow-up" button that uses the `mailto:` functionality.
